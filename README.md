## Instalation



## Step 1 Download And Install 

- composer install
- npm install
- cp .env.example .env
- php artisan key:generate

## Step 2:- Configure Database

- DB_CONNECTION=mysql 
- DB_HOST=127.0.0.1 
- DB_PORT=3306 
- DB_DATABASE=DB_NAME
- DB_USERNAME=DB_USER
- DB_PASSWORD=DB_PASSWORD

##  Migrate the database

php artisan migrate

## Seed the database

- php artisan db:seed

## Run Laravel Vue IN (run command in separate terminal)

- npm run watch
- php artisan serve

## OPEN BROWSER

- Open the URL in the browser: http://127.0.0.1:8000
